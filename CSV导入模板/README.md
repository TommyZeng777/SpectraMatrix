# CSV 导入模板

这里放的是给新实验照着填的最小 CSV 表格结构。

## 必需文件

1. `01_光谱矩阵模板.csv`
   - 每一行是一条光谱。
   - 必须包含 `sample_id`。
   - 必须包含连接列，推荐命名为 `sample_link_code`。
   - 后面的数字列是波长，例如 `500,501,502`。

2. `02_监督标签模板.csv`
   - 每一行是一个监督标签。
   - 必须包含和光谱表一致的连接列，推荐命名为 `sample_link_code`。
   - 至少包含 `ppm_mg_kg`。
   - 如果已有分类标签，可以额外提供 `high_risk_gt500_label` 和 `three_class_label`。

## 最小约定

- 光谱表和监督表通过 `sample_link_code` 连接。
- 用户不需要写 JSON；平台会在导入后自动生成内部配置。
